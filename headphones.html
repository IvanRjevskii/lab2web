Подробное описание товара

Модель Wireless Pro обеспечивает кристально чистый звук и глубокие басы. Система активного шумоподавления позволяет сосредоточиться на музыке.

  1. Поддержка кодеков AAC и SBC для качественного звука
  2. Встроенный микрофон для звонков и голосовых помощников
  3. Складная конструкция для удобной переноски
  4. Быстрая зарядка (10 мин = 3 часа работы)
  5. Мягкие амбушюры для комфортного ношения
  6. Кнопки управления на корпусе

Комплектация

В коробке вы найдете: сами наушники, чехол для переноски, кабель USB Type-C для зарядки и аудиокабель 3.5 мм для проводного подключения.
